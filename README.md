## 单词积累助手 (Vocabulary Manager)

一个基于 Python 和 tkinter 的图形化单词/词组管理工具，采用 文件夹树 + SQLite 数据库 管理词汇，支持模糊搜索、Excel 导入导出、重复清理等功能。

### ✨ 主要功能

📁 文件夹树管理：以树形结构组织单词，支持新建、重命名、删除、移动文件夹。

📝 单词词条管理：每个文件夹下可存放多个“单词-释义”对，支持增、删、改、移动。

🔍 多关键词模糊搜索：在当前文件夹及所有子文件夹中，按 OR 逻辑模糊搜索单词和释义。

📥 Excel 导入：从三列 Excel（文件夹名/单词/释义，无表头）批量导入单词，自动创建文件夹。

📤 Excel 导出：将所有单词导出为三列 Excel（文件夹名/单词/释义，无表头），方便备份或二次加工。

🧹 一键去重：自动清除同一文件夹内单词和释义完全相同的重复词条。

💾 多库支持：可新建/打开多个 .db 数据库文件，满足不同词汇场景（如不同语言、不同考试）。

### 🖥️ 运行环境

Python 3.7 及以上

tkinter（通常已内置，Linux 需额外安装）

可选依赖（用于 Excel 和主题美化）：

openpyxl：支持 Excel 导入导出

ttkthemes：提供现代暗色/亮色主题

### 📦 安装

克隆或下载本项目代码到本地。

安装必需依赖（Excel 支持）：

```bash
pip install openpyxl
```

(可选) 安装主题美化库：

```bash
pip install ttkthemes
```

如果运行在 Linux 且缺少 tkinter：

```bash
sudo apt update
sudo apt install python3-tk
```
### 🚀 快速开始
进入项目目录，运行：

```bash
python main.py
```

首次启动会提示新建或打开一个数据库文件（.db）。选择新建后，程序会在指定位置创建空库并进入主界面。

左侧为文件夹树（根节点“词汇库”），右侧为选中文件夹内的单词列表。

通过工具栏或右键菜单进行管理。

### 📖 使用说明

文件夹操作

```
新建子文件夹：选中文件夹 → 右键“新建子文件夹”或工具栏“新建文件夹”。

重命名：右键“重命名”或工具栏“重命名”。

移动：右键“移动”（仅可移动到父文件夹或子文件夹）。

删除：删除文件夹时会同时删除其下所有子文件夹和单词，有确认提示。
```

单词操作

```
新建单词：选中文件夹 → 右键“新建单词”或工具栏“新建单词”。

编辑单词：双击表格行或右键“重命名”。

移动单词：右键“移动”到其他文件夹。

删除单词：支持多选后批量删除。
```

搜索

```
在顶部搜索框输入关键词（支持多个，空格分隔），点击“搜索”。
程序将检索当前文件夹及其所有子文件夹中，单词或释义包含任一关键词的词条，并在新窗口显示结果。双击结果可快速跳转到对应文件夹。
```

导入导出

```
Excel 导入：文件 → 从Excel导入…
选择三列 Excel 文件（无表头）：

第1列：直接文件夹名（空 = 根文件夹）

第2列：单词

第3列：释义
若文件夹不存在则自动创建，重复单词正常添加。

Excel 导出：文件 → 导出到Excel…
将所有单词按同样三列格式导出，无表头。
```


清除重复
```
文件 → 清除重复单词
自动扫描所有文件夹，删除（文件夹内）单词与释义完全相同的重复项，仅保留最早添加的一条。
```

### 🗂️ 项目结构
```text
vocabulary_manager/
├── main.py                 # 程序入口
├── db.py                   # SQLite 数据库操作封装
├── ui/
│   ├── __init__.py
│   ├── main_window.py      # 主窗口布局、菜单、工具栏
│   ├── folder_tree.py      # 左侧文件夹树组件
│   ├── entry_list.py       # 右侧单词列表组件
│   ├── dialogs.py          # 重命名、编辑、移动对话框
│   └── search.py           # 搜索结果窗口
└── last_db.txt             # 自动生成，记录上次打开的数据库路径
```

### 🎨 主题美化

程序默认使用 tkinter 内置 clam 主题（较现代）。
若安装了 ttkthemes，可在 main_window.py 中切换为暗色主题，例如：

```python
from ttkthemes import ThemedTk

class MainWindow(ThemedTk):
    def __init__(self):
        super().__init__()
        self.set_theme("equilux")   # 暗色主题

```
